# Free Movies

TMDB for metadata, vidsrc.to for playback.

## Embed URL format (vidsrc.to)

```
Movie:   https://vidsrc.to/embed/movie/{tmdb_id}
Season:  https://vidsrc.to/embed/tv/{tmdb_id}/{season}
Episode: https://vidsrc.to/embed/tv/{tmdb_id}/{season}/{episode}
```

## Files

```
index.html   — markup
style.css    — dark theme
app.js       — all logic
```

Open index.html directly in browser. No build step.
